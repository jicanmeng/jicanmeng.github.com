centos的一些问题

作者：jicanmeng

时间：2014年11月2日

  1. 如何查询你电脑上的centOS的版本
  2. 如何查询linux的版本
  3. 从哪里可以下载对应centOS对应版本的linux kernel代码
  4. centos下如何切换拼音输入法
  5. centos每次进入一个目录但同时打开一个新的窗口的问题的解决

1. 如何查询你电脑上的centOS的版本？

通过 /etc/issue 文件，这个文件是控制台界面，用来登录时的提示信息的，而这个提示信息就有CentOS的版本。

2. 如何查询linux的版本？

通过uname -r命令。

3. 从哪里可以下载对应centOS对应版本的linux kernel代码?

经过cat /etc/issue和uname -r，我知道我的 CentOS的发布版本是 6.5，内核版本是2.6.32-431.3.1。源代码的官网：http://vault.centos.org/。进入官网后，依次是进入 6.5/，进入os/，进入Source/，进入SPackages/，找到kernel-2.6.32-431.el6.src.rpm，下载就行了。

在官网上也可以下载对应版本的iso镜像文件。

4. centos下如何切换拼音输入法?

centos6.5自带有拼音输入法，但是默认情况下是不打开的。通过下面的方式打开：

  1. System->Preferences->Input Method，跳出一个对话框。
  2. 勾选上“Enable input method feature”。
  3. 点击“Input Method Preferences”，跳出一个“IBus Preferences”对话框。
  4. 切换到“Input Method”标签。
  5. 点击“Select an input method”下拉框，选择“Chinese->pinyin”后，点击“Add”按钮，再点击“Close”按钮。返回到刚才到对话框，点击“Close”按钮退出。
  6. 打开浏览器，按“ctrl+空格键”，就可以输入中文了。

5. centos每次进入一个目录但同时打开一个新的窗口的问题的解决?

勾选 Edit -> Preferences -> Behavior -> Always open in browser windows 即可解决。
