Git 概述

作者：jicanmeng

时间：2014年08月25日

Git是一个免费的、开源的、分布式的版本控制工具。经常用于源代码的管理。

<<Pro Git>>和<<Git权威指南>>是两本非常好的书。下面几篇文章总结了Git的一些简单的知识点，主要依据这两本书以及网上的一些教程。

Git的相关内容如下所示：

  1. 如何对git进行配置--"git config"
  2. git的基本命令--"git init;git add;git commit;git log;git show;git diff;git status"
  3. git对象--"git cat-file"
  4. 如何创建分支？如何切换分支？--"git branch;git checkout"
  5. 如何对版本库进行回退?--"git reset"
  6. 如何进行提交之间的合并? --"git merge"和"git rebase"
  7. 如何使用git打patch？--"git format-patch;git am;git apply"
  8. 如何使用git挑选其它分支上面的commit?--"git cherry-pick"

  10. git的远程仓库操作命令有哪些?
