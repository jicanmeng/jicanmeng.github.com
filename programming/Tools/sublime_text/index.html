sublime text 3 简单用法

作者：jicanmeng

时间：2017年02月14日

sublime text 是一款编辑器，相比较nootepad++，它能直接打开一个目录，而且在左侧直接显示目录的结构。看起来很酷。

前面几天，上网时在网易云课堂中偶然看到了十八哥的教程《布尔教育：8小时学会HTML网页开发》,视频中使用的就是sublime text这款编辑器。而且作者操作起来看着很流畅，于是下载下来看一看。使用的时候需要安装几个插件，如下:

1. 支持中文输入

这需要安装一个InputHelper插件。安装步骤如下：

  1. cd ~/.config/sublime-text-3/Packages
  2. git clone https://github.com/xgenvn/InputHelper.git
  3. 重启sublime text。
在使用的时候，如果要输入中文，步骤如下：
  1. Ctrl+Shift+Z 调出输入框（虽然不习惯也不爽，但能支持中文输入已经不错啦_^)
  2. 输入框窗口输入中文，然后按下Enter或者Ctrl+Enter键输入文字到sublime text中了。

2. 编辑html文件时快速生成html头部信息：

这需要安装一个名称为emmet的插件。安装步骤如下：

  1. cd ~/.config/sublime-text-3/Packages
  2. git clone https://github.com/sergeche/emmet-sublime.git
  3. 重启sublime text。

有一点需要注意：

WARNING: When plugin is installed, it will automatically download required PyV8 binary so you have to wait a bit (see Loading PyV8 binary message on status bar). If you experience issues with automatic PyV8 loader, try to install it manually.

在刚开始编译一个后缀名为html的文件时，输入符号，再按Tab键，就可以展开成为一段代码。符号如下：
  • html:5：HTML5类型
  • html:4sHTML4严格文档类型
  • html:4tHTML4过渡文档类型
  • html:xsXHTML严格型文档类型
  • html:xtXHTML过渡型文档类型
这样敲代码的时候感觉还是很爽的！

3. 其它一些有用插件：

  • 输入javascript代码时, 能够自动补齐：SublimeCodeIntel.
  • 可以检测一行的后面是否有多余的空白字符：TrailingSpaces.
  • 可以高亮匹配的括号：BracketHighlighter.
安装的时候，首先按住ctrl+shift+p调出一个下拉框，选择package Control: Install Package, 回车。然后输入上面的插件的名称，就会进行安装了，安装的状态可以在左下角的状态栏中看到。
