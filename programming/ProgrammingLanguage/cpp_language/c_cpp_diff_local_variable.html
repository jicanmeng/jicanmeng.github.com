局部变量的定义的地方

作者：jicanmeng

时间：2014年11月26日

c语言的函数中，所有的局部变量的定义必须放在函数开始的地方。而在c++中则可以在使用此变量之前再进行定义。

看一个示例程序： 

#include <iostream>

int main()
{
    int a = 0;
    a = 3;

    int b = 0;
    b = 3;

    std::cout << "a is " << a <<
	    ", b is " << b << std::endl;
    return 0;
}
