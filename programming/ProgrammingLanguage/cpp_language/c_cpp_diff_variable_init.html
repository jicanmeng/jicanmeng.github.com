变量的初始化

作者：jicanmeng

时间：2014年11月26日

在变量定义的同时给变量赋初值，称为变量的初始化。c语言中变量的初始化是通过赋值运算符"="来完成的，c++中则有两种形式，可以通过赋值运算符"="，也可以通过括号"()"。

看一个示例程序： 

#include <iostream>

int main()
{
    int a = 3;
    int b(4);
    std::cout << "value of a is " << a <<
	    ", value of b is " << b << std::endl;
    return 0;
}
